Игра zuma.
Доступны несколько уровней. Цель- уничтожить все шарики.
На каждом уровне шары двигаются по кривым.
Вращение лягушки осуществляется при помощи мышки. (старайтесь не попадать курсором на QLabel так как там не считываются координаты курсора)
Стрелять на Space.
Поменять предыдущий и последующий шары местами на Shift.
Если к концу пути хотя бы один шар остался, то присуждается поражение.

Добавлено корректное удаление шариков.
Для уровней теперь есть класс лоигки, а все уровни задаются параметрами
Добавлено главное меню, меню выбора уровней.

Для того чтобы начать игру, нужно ввести имя.
Если у вас есть незаконченная игра, то вы можете загрузить сохранение, нажав Saves
Далее надо нажать Play.
Либо вы можете начать новую игру, нажав Levels и далее выбрать уровень.
Сохраняются только незаконченные игры. Автосохранение происходит при выходе из игры.


Уровни:
1) Движение по синусоиде
2) Движение по спирали
3) Движение по окружности
4) Движение по элиптической лемнискате
5) Движение по кривой, заданной параметрически

В итоге выводится таблица результатов. (топ 5 игроков)
